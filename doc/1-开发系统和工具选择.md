# 系统选择
  - windows 
    * 价格亲民
    * 无法开发ios
  - mac
    * 价格较高
    * 既可以开发ios，也可以开发android
  > 推荐使用mac

# 工具选择
  - Visual Studio Code
    * 前端开发利器
    * 无法开发android，如果要开发，还需要安装android studio
  - android studio
    * android开发利器
    * google亲儿子。由于flutter和android都是谷歌开发的，因此android对一些新特性支持比较好，开发和调试的体验比vscode要成熟。
  > 推荐使用android开发
